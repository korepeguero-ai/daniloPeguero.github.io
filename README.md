## About this site

This site serves as my online resume — a quick, shareable way for recruiters, collaborators, and anyone curious to learn about my background, skills, and experience.

## Built with

- HTML / CSS
- GitHub Pages

## Updating the resume

The resume content lives in `index.html`. To make changes:

1. Edit `index.html` directly on GitHub, or clone the repo locally.
2. Commit your changes to the `main` branch.
3. Changes go live at `https://daniloPeguero.github.io` within a few minutes.
